# medusa-simulacao-bruteforce
Estudo técnico e documentado sobre auditoria de segurança com Medusa, abordando ataques de força bruta e password spraying em serviços comuns. O repositório inclui wordlists, comandos reais, explicações, análises e recomendações de defesa. Ideal para demonstrar conhecimentos em testes de intrusão e segurança ofensiva.
